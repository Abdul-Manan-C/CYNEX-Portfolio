# **App Name**: CYNEX - Command Center 2026

## Core Features:

- Cybernetic UI Design: Implementation of a futuristic, terminal-inspired user interface with glassmorphism and neon accents.
- Live Telemetry Panel: A floating panel that displays real-time system information such as IP address, device type, OS platform, and GPU renderer.
- Interactive Terminal: A functional terminal emulator with a command database for executing various system commands and a Linux mastery quiz.
- Cyber Intrusion Simulator: An interactive game simulating a cyber intrusion scenario, where users can practice network scanning, enumeration, and exploitation techniques.
- C++ Project Showcase: A dedicated section to showcase a C++ bus reservation system project, complete with source code, architecture overview, and local execution instructions.
- Animated Backgrounds: Dynamic matrix rain and grid backgrounds to create an immersive cybernetic atmosphere.
- Section Visibility Observer: Implementation of an Intersection Observer to smoothly transition sections into view as the user scrolls.
- 3D Card Hover Effect: Interactive glass panels that react to pointer movement, providing a subtle 3D tilt and glow effect.

## Style Guidelines:

- Primary neon colors: #00d4ff (blue), #00ff41 (green), #ff3e3e (red), #b026ff (purple) for a futuristic aesthetic.
- Background color: #05070a (bg-black) for a dark, immersive environment.
- Glass effect: rgba(10, 15, 25, 0.85) with a border of rgba(0, 212, 255, 0.2) for glass panels.
- Headline font: 'Space Grotesk' (sans-serif) for impact and a contemporary look.
- Body and code font: 'JetBrains Mono' (monospace) for a technical, readable experience.
- Font Awesome icons for a consistent and recognizable visual language.
- Grid-based layout for structured content presentation and responsive design.
- Subtle transitions and hover effects to enhance interactivity and user engagement.
- Mobile-first approach with responsive design to ensure optimal viewing experience across devices.
